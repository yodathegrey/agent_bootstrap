## Format

Each ADR follows a consistent structure:

- **Title** -- short descriptive name
- **Date** -- date the decision was recorded
- **Status** -- Proposed, Accepted, Deprecated, or Superseded
- **Context** -- the forces and circumstances that led to this decision
- **Decision** -- the change or choice that was made
- **Consequences** -- the resulting effects, both positive and negative
